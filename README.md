### blog
Harrison's hexo 博客仓库
用户存放 hexo 相关配置，不包含博客内容，具体博客内容可查看 [blog-content](https://github.com/KangHarrison/blog-content)

### How to use it
在服务器上拉取本仓库，之后再博客目录下执行：
> `hexo clean`
>
> `hexo g`
> 
> `hexo s`

这样就就可以通过服务器的ip+port访问。eg:192.168.0.1:4000

tips:需要确保服务器的4000端口是可访问的。

有关博客搭建的更多消息，可查看：[Hexo博客搭建2.0](http://harrison-hub.cn/2022/05/10/Hexo%E6%90%AD%E5%BB%BA%E5%8D%9A%E5%AE%A22.0/)
